Review Proposed Tags

The approval process for tags is designed to keep the tags list from growing exponentially. Tags that are candidates for approval are those that can apply more than a single idea. Edge-case tags (likely to be used very rarely) are not good at relating similar tags and do not help the system.

Try to avoid approving tags that could be considered synonyms for existing tags.
